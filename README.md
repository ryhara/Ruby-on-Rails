## Usage
```
$ bundle install
$ rails webpacker:install
$ rails webpacker:compile
$ rails db:migrate
```
```
$ rails s
```
If "Webpacker::Manifest::MissingEntryError" occur
```
$ yarn add @babel/plugin-proposal-private-methods @babel/plugin-proposal-private-property-in-object
```
